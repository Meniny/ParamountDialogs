## Introduction

`ParamountDialogs` is a delightful dialog view for iOS in Swift.

![https://i.loli.net/2018/01/16/5a5dc2a655189.jpg](https://i.loli.net/2018/01/16/5a5dc2a655189.jpg)

## Requirements

* iOS 8.0+
* Xcode 9 with Swift 4

## Installation

#### CocoaPods

```ruby
use_frameworks!
pod 'ParamountDialogs'
```
